## 💻 Cara Menjalankan Kode

1. **Prasyarat:**
   - Web Browser modern (Google Chrome, Firefox, MS Edge, atau Safari).
   - Editor teks (misal: Visual Studio Code).
   - Ekstensi **Live Server** di VS Code (opsional, direkomendasikan).

2. **Membuka File HTML:**
   - Klik kanan pada file `.html` mana saja (misal `CSS/Flexbox.html` atau `JS/Todo_App.html`).
   - Pilih **Open with Live Server** (atau buka langsung file tersebut di browser).

3. **Menjalankan File JavaScript:**
   - File JavaScript dapat dilihat output-nya melalui DevTools Browser (Tekan `F12` atau `Ctrl + Shift + I` lalu buka tab **Console** pada file HTML yang terhubung).
   - Atau bisa dijalankan menggunakan Node.js di terminal:
     ```bash
     node JS/Arrays.js
     ```
